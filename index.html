A project that took a PSD and was built into a fully functioning web application.

Front-End Functional Requirements:

  • Use HTML (EJS for JS templating), CSS, and Javascript to build out site.
  • Chop PSD's to access images and build each page structure/design with HTML/CSS
  • Add Javascript functionality to static design to create enjoyable UX.

Back-End Functional Requirements:

  • Use Node.JS for server-side and MongoDB for database.
  • Create admin dashboard for restaurant personnel to Access
  • Require user authentication to allow personnel to create/edit/delete menus and menu items.
  • Create user model, store users in database, track sessions on front-end, and hash passwords
  • Create menu data Model, store data in database, and retrieve filtered data to display on menu pages.
  • Allow customer reservations.
  • Create reservation data model, store data, send response trigger to customer to confirm reservation.
