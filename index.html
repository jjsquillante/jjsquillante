My Work at BBVA

Most of my time at BBVA has been spent improving the workflow for our financial analysts in order to enhance the reporting process. Initially, this started as writing a suite of scripts (Node.js) to automate manual tasks and evolved into a web application that runs reports and displays reporting trends/stats.

Web Application Features

*The entire application was built by myself, from scratch, using the tools above.*

Report Generation

- The report generation process enables analysts to run reports, track their progress, and automatically post any results/controls/documentation to the google drive and JIRA. (GIF 1/2)

File Upload

- Analysts are able to initiate the ETL process by loading external data into tables within the database for reporting purposes. (GIF 1)

- An automated email is submitted with the results of the file load. A file log is created and stored within the google drive. (GIF 2)

Reporting Dashboard

- The reporting dashboard provides insight for analysts to determine when the monthly data is ready to run their reports.

- Also, charts are provided to display reconciliation and trends analysis.
